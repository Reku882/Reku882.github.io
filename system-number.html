ทฤษฎีบทของระบบจำนวนจริงที่เกี่ยว
ข้องกับการบวกและการคูณ

ทฤษฎีบท 1 กฎการตัดออกสำหรับการบวก ให้ a, b, c เป็นจำนวนจริงจะได้
(1) ถ้า a + c = b + c แล้ว a = b
(2) ถ้า a + b = a + c แล้ว b = c
ทฤษฎีบท 2 กฎการตัดออกสำหรับการคูณ ให้ a, b, c เป็นจำนวนจริงจะได้
(1) ถ้า ac = bc และ c ≠ 0 แล้ว a = b
(2) ถ้า ab = ac และ a ≠ 0 แล้ว b = c
ทฤษฎีบท 3 ให้ a เป็นจำนวนจริง จะได้ a x 0 = 0
ทฤษฎีบท 4 ให้ a เป็นจำนวนจริง จะได้ (-1) x a = -a
ทฤษฎีบท 5 ให้ a, b เป็นจำนวนจริง จะได้ว่า ab = 0 ก็ต่อเมื่อ a = 0 หรือ b = 0
ทฤษฎีบท 6 ให้ a, b เป็นจำนวนจริงจะได้ว่า
(1) a(-b) = -ab
(2) (-a)b = -ab
(3) (-a)(-b) = ab
ทฤษฎีบท 7 ให้ a, b, c เป็นจำนวนจริงจะได้ว่า
(1) a(b – c) = ab – ac
(2) (a - b)c = ac – bc
ทฤษฎีบท 8 ให้ a เป็นจำนวนจริง ถ้า a ≠ 0 แล้ว a^-1 ≠ 0

การลบและการหารจำนวนจริง

บทนิยามการลบ ให้ a, b เป็นจำนวนจริง a ลบด้วย b เขียนแทนด้วยสัญลักษณ์ a – b โดยที่จะได้นิยาม a – b คือผลบวกของ a กับตัวผกผันของการบอก b นั้นเอง
บทนิยามการหาร a, b เป็นจำนวนจริง a หารด้วย b เขียนแทนด้วยโดยที่จะได้นิยามคือผลบอกของ a กับตัวผกผันการคูณของ b นั่นเอง
